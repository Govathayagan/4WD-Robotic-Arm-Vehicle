#  4WD Robotic Arm Vehicle 

## 🔹 Description

This project is a 4WD robotic vehicle integrated with a 5-DOF robotic arm, controlled using ESP32 via Bluetooth. It is designed for object pickup, transportation, and disposal applications, reducing human effort in hazardous environments.

## 🔹 Features

* 4WD vehicle movement control
* 5-DOF robotic arm control
* Bluetooth mobile app control
* Real-time operation
* Multi-functional system

## 🔹 Components Used

* ESP32
* L298N Motor Driver
* DC Motors (4x)
* Servo Motors (5x)
* Battery
* Robotic Arm Kit

## 🔹 Working Principle

The system is controlled using a mobile application via Bluetooth. Commands are sent to the ESP32, which processes them and controls the DC motors for movement and servo motors for arm operation.

## 🔹 Applications

* Garbage collection systems
* Industrial automation
* Hazardous material handling
* Robotics research

* Add IoT control (WiFi)
